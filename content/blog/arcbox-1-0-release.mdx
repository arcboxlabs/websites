## We Did It!

After 18 months of intensive development and feedback from hundreds of beta testers, we are proud to announce **ArcBox Desktop 1.0**.

## What's New

ArcBox Desktop 1.0 brings:

- **Full Docker Compatibility** - Run your existing Docker Compose files without modification
- **Apple Silicon Native** - Optimized for M-series chips with 4x faster startup than Docker Desktop
- **Open Source** - The entire codebase is available for inspection and contribution
- **Production Ready** - Used in production by dozens of companies

## Performance

Compared to Docker Desktop on Apple Silicon, ArcBox delivers:

- 4x faster container startup times
- 30% lower memory overhead
- Smoother performance under load

## Getting Started

Head over to [arcbox.dev](https://arcbox.dev) to download ArcBox Desktop 1.0 today.
